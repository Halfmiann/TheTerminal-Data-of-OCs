# OC世界观时间轴网站 - 需求拆解文档

## 产品概述

- **产品类型**: 个人OC世界观展示网站
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: OC创作者及其粉丝、世界观爱好者
- **核心价值**: 以极简未来感的时间轴形式，沉浸式呈现原创角色世界观的时间线故事
- **界面语言**: 中文
- **主题偏好**: 浅色（白色为主，黑白灰基调）
- **导航模式**: 路径导航
- **导航布局**: Topbar（顶部导航）
- **设计风格参考**: Cytus α 界面风格——极简、未来感、几何线条、干净排版、高级科技感

## 页面布局建议

### 首页/封面
- **布局模式**: 单栏居中布局，全屏Hero式设计
- **视觉重心**: 世界观标题 + 进入按钮 —— 首屏即核心视觉焦点
- **结果承载区**: 无（纯展示与入口引导
- **设计要点**: 大字号标题、几何装饰线条、微动效入场动画、底部"进入时间轴"CTA按钮

### 时间轴主页
- **布局模式**: 垂直居中时间轴 + 左右交替节点卡片
- **视觉重心**: 时间轴中心线 + 节点卡片 —— 时间线为核心交互载体
- **结果承载区**: 节点展开预览（hover/点击展开故事简介）
- **设计要点**: 垂直时间轴贯穿页面，节点沿轴线分布，hover时卡片微动效放大，点击后平滑过渡到详情页

### 故事详情页
- **布局模式**: 单栏文档流布局，左侧/顶部有返回按钮
- **视觉重心**: 故事正文内容 —— 阅读体验优先
- **结果承载区**: 故事内容区（标题、时间、角色、正文）
- **设计要点**: 干净的排版、角色卡片侧栏、返回时间轴按钮、入场过渡动画

### 世界观总览页
- **布局模式**: 单栏分节布局
- **视觉重心**: 世界观设定内容 —— 信息展示为主
- **结果承载区**: 设定卡片/章节内容
- **设计要点**: 分章节展示世界观设定、几何线条分隔、滚动渐入动效

## 功能列表

### 首页/封面
- **页面目标**: 展示世界观标题与简介，引导用户进入时间轴
- **功能点**:
  - **世界观标题展示**: 大字号主标题 + 副标题，配合几何线条装饰，入场时有淡入+位移动画
  - **简短介绍**: 1-2 段世界观核心简介，打字机/渐入效果
  - **进入时间轴按钮**: 主 CTA 按钮，hover 有发光/线条延展动效，点击跳转至时间轴页
  - **背景装饰**: 极简几何线条/网格背景，缓慢呼吸/流动动效

### 时间轴主页
- **页面目标**: 以时间轴形式展示所有关键时间节点，支持交互浏览
- **功能点**:
  - **垂直时间轴渲染**: 居中垂直线条，节点沿轴线左右交替排列，展示时间、标题、简介
  - **节点 hover 微动效**: hover 时节点圆点放大、卡片轻微上浮、线条高亮
  - **节点点击跳转**: 点击节点卡片平滑过渡到对应故事详情页
  - **滚动渐入效果**: 滚动时节点卡片从两侧滑入，带视差感
  - **时间轴进度指示**: 当前滚动位置对应的时间节点高亮指示

### 故事详情页
- **页面目标**: 展示单个时间点的完整故事内容、相关角色与事件描述
- **功能点**:
  - **故事内容展示**: 标题、时间标签、正文内容，干净的阅读排版
  - **相关角色卡片**: 侧边/底部展示该事件涉及的角色（头像、名字、简短介绍）
  - **返回时间轴按钮**: 左上角返回按钮，点击平滑过渡回时间轴页
  - **入场过渡动画**: 从时间轴节点位置扩散/淡入的页面过渡效果

### 世界观总览页
- **页面目标**: 整体世界观设定的概述展示
- **功能点**:
  - **设定分节展示**: 世界背景、势力划分、核心概念等分章节展示
  - **滚动渐入动效**: 各章节随滚动依次淡入出现
  - **几何线条装饰**: 章节间用几何线条/图形分隔，保持风格统一

## 动效与交互设计规范

> **设计原则**: 简洁克制，符合 Cytus α 极简未来感风格

| 动效类型 | 应用场景 | 效果描述 |
|---------|---------|---------|
| 页面过渡 | 路由切换 | 淡入淡出 + 轻微位移，时长 300-400ms，缓动函数 ease-out |
| 节点 hover | 时间轴节点 | 圆点放大 1.2x，卡片上浮 4px，边框高亮 |
| 滚动渐入 | 页面滚动 | 元素从透明度 0 → 1，位移 20px → 0，交错出现 |
| 按钮 hover | 所有按钮 | 背景色过渡 + 边框线条延展/发光效果 |
| 背景动效 | 首页背景 | 几何线条缓慢呼吸/流动，极低速率 |
| 时间轴进度 | 时间轴页滚动 | 轴线进度填充，当前节点高亮 |

# UI 设计指南

## 1. 设计推导依据

- **参考意图**: Mood Reference —— 参考 Cytus α 的极简未来感气质、几何线条与干净排版；用户概念图无法加载，按描述的黑白灰未来感方向自由构建。
- **核心情绪 / 应用类型**: 叙事型内容站，通过时间轴探索 OC 世界观，情绪为冷静、精密、科幻诗意。
- **独特记忆点**: 中央细线时间轴 + 节点几何脉冲微动效 + 页面切换时的横向滑动与淡入淡出，模拟「读取记忆档案」的未来终端感。

## 2. Art Direction

- **方向名**: Mono Future Archive
- **Design Style**: Swiss Minimalist + Geometric Tech —— 以瑞士极简排版为骨架，叠加几何线条与细线装饰，契合 Cytus α 式的干净未来感。
- **DNA 参数**: 圆角 sharp（`rounded-none`）/ 阴影 none 至 subtle / 间距 spacious（`gap-6`/`p-8`）/ 字体方向：无衬线几何 + 等宽数字 / 装饰手法：细直线、圆点节点、细边框、几何分隔符。
- **应用类型**: Content / Narrative —— 以阅读与探索为主，大留白、强排版节奏。

## 3. Color System

**色彩关系**: 纯白背景 + 近墨黑正文 + 极浅灰分隔与卡片底 + 单一冷灰主色承担交互与激活态，整体黑白灰单色系，通过明度层级建立秩序。
**配色设计理由**: 以白色为主色调呼应「洁净未来感」；primary 选用冷调深灰而非纯黑，保留科技精密感；accent 用极浅灰承接 hover 与选中，避免高饱和色破坏极简氛围。
**主色推导**: 从 Cytus α 的黑白界面与用户要求的「黑白灰基调 + 未来感」出发，primary 取冷调深灰（偏蓝相极低），用于 CTA、时间轴激活节点与关键链接，兼具识别度与克制感。
**使用比例**: 70% 中性白 / 25% 灰阶辅助 / 5% primary；primary 仅用于主行动、当前节点与品牌锚点，不与 tab 底色、边框、图标全部叠加。

| 角色 | CSS 变量 | Tailwind Class | HSL 值 | 设计说明 |
|---|---|---|---|---|
| bg | `--background` | `bg-background` | hsl(0 0% 100%) | 纯白页面背景，未来感基底 |
| card | `--card` | `bg-card` | hsl(0 0% 98%) | 极浅灰卡片，微浮起感 |
| text | `--foreground` | `text-foreground` | hsl(220 8% 12%) | 近墨黑正文，高对比可读 |
| textMuted | `--muted-foreground` | `text-muted-foreground` | hsl(220 5% 45%) | 中灰辅助文字、时间戳、说明 |
| primary | `--primary` | `bg-primary` / `text-primary` | hsl(220 10% 20%) | 冷调深灰主色，CTA 与激活节点 |
| primaryForeground | `--primary-foreground` | `text-primary-foreground` | hsl(0 0% 100%) | 主色上的纯白文字 |
| accent | `--accent` | `bg-accent` | hsl(220 7% 94%) | 浅灰 hover / 选中 / 骨架屏底 |
| accentForeground | `--accent-foreground` | `text-accent-foreground` | hsl(220 8% 18%) | accent 上的深灰文字图标 |
| border | `--border` | `border-border` | hsl(220 6% 88%) | 细线分隔、卡片边界、输入框 |

**语义色提示**: 无强语义色需求；若需成功/警告/错误，统一降低饱和度至 10%-15% 区间，色温对齐冷灰：成功 hsl(160 12% 35%) 三态 bg hsl(160 20% 95%) / border hsl(160 10% 80%) / text hsl(160 12% 30%)；警告 hsl(38 15% 40%)；错误 hsl(0 12% 45%)，均保持克制不抢戏。

## 4. 字体与节奏

- **font-display**: Space Grotesk —— 几何未来感，契合 Cytus α 式科技极简气质，用于大标题与时间数字。
- **font-body**: Noto Sans SC —— 中文正文清晰中性，长文阅读舒适，与 Space Grotesk 的冷调几何感互补。
- **字号**: H1 text-5xl md:text-6xl；H2 text-2xl md:text-3xl；body text-base；muted text-sm；时间数字用等宽特征（tabular-nums）。
- **圆角**: 极小 / sharp —— 全用 `rounded-none` 或 `rounded-sm`，强化几何线条与精密感。

## 5. 全局布局契约

- **Reference Layout Use**: 按需求结构推导，参考 Cytus α 的居中对称、大留白与细线分隔气质。
- **Page / Section Order**: 首页封面 → 时间轴主页 → 故事详情页 → 关于/世界观总览页，与需求 1:1 对齐。
- **Standard Content Zone**: Landing/Content 级 `max-w-5xl` + `mx-auto`；时间轴页可放宽至 `max-w-4xl` 保持居中聚焦。
- **Shell / Frame Alignment**: 内容容器与框架同宽，顶部极简导航条与内容区对齐，无侧边栏。
- **Padding & Rhythm**: `px-4 md:px-6 lg:px-8 py-16 md:py-24`，垂直节奏以 `space-y-12` / `space-y-16` 为单位。
- **Full-bleed Zones**: 首页封面背景与几何装饰可全宽，标题与入口按钮仍受 `max-w-5xl` 约束。
- **Local Narrowing**: 故事详情正文收窄至 `max-w-2xl` 居中，保证阅读行长。
- **Overflow Strategy**: 时间轴节点横向列表（移动端）使用 `overflow-x-auto`；详情页长文自然纵向滚动。
- **Flexibility Boundary**: 允许移动端 padding、卡片间距与节点尺寸微调；不允许改变主色、圆角、细线语言与居中对称布局。

## 6. 视觉与动效

- **装饰**: 细直线、圆点节点、几何十字/刻度线、极细边框；大面积留白。
- **阴影/边界**: 无 —— 全部用边界线与背景色差区分层级，拒绝投影。
- **动效**: 克制精致 —— 节点 hover 为圆点外扩 + 细线延伸；页面切换为横向位移 + 淡入（300-400ms ease-out）；滚动渐入用 opacity + translateY 8px；所有动效 cubic-bezier(0.22, 1, 0.36, 1)。

## 7. 组件原则

- 按钮：主按钮实心深灰 + 白字；次按钮细边框 + 深灰字；ghost 按钮仅文字下划线或左侧竖线标记。
- 时间轴节点：默认空心圆 + 细线连接；hover 圆外扩 2px 并填充；active/当前节点实心圆 + 横向短线延伸。
- 卡片：极浅灰底 + 1px 细边框 + 无圆角；hover 边框色加深至 textMuted。
- 导航：顶部细线分隔，文字链接 + 下划线激活态；focus-visible 用 2px outline 偏移 2px。
- 加载与空状态：用细线条骨架 + 几何占位图形，延续同一语言。

## 8. Image Direction

- **Image Role**: 章节头图 / 故事氛围图 —— 每个时间节点配一张低信息权重的氛围图，置于故事详情页顶部或时间轴节点旁，辅助叙事而非喧宾夺主。
- **Image Art Direction**: 黑白或单色调极简科幻场景；构图以几何建筑、空旷空间、光点阵列、地平线线条为主；冷白光、柔和漫射、高对比阴影；材质为光滑金属、哑光混凝土、玻璃与雾气；情绪冷静、神秘、带有诗意的孤独感；图像整体偏亮（白底友好），可叠加细噪点或扫描线质感呼应未来终端。
- **Image Prompt Keywords**: monochrome sci-fi landscape, minimalist geometry, empty futuristic architecture, soft cold lighting, high contrast shadows, matte concrete and glass, foggy atmosphere, horizon line, light dots array, cinematic composition, clean negative space, slight film grain
- **Image Avoidance**: 避免彩色霓虹、赛博朋克杂乱招牌、人物特写、卡通插画、3D 渲染塑料感、过度复杂的机械细节、典型 AI 素材库式「未来城市天际线」。

## 9. Anti-patterns

- **Split personality**: 首页用纯白极简、详情页突然变深色或加渐变；全站保持白底灰线同一系统。
- **Phantom tokens**: 编造不存在的 CSS 变量；所有颜色与间距从已定义 token 取用。
- **Default SaaS drift**: 回到默认蓝色按钮、紫色渐变卡片；用冷深灰 + 细线语言贯穿。
- **Invisible interaction**: 只做 hover 动效忽略 focus-visible；所有可交互元素必须有可见键盘焦点。
- **Mono-hue tyranny**: 把深灰同时用在按钮、tab、图标、边框、链接；primary 只给 CTA 与当前节点，其余交给 accent / border / textMuted。
- **Animation bloat**: 每个元素都加动效导致页面轻浮；仅节点 hover、页面切换、滚动渐入三处使用动效，其余保持静态。
